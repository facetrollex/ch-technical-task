Basic CS
Tech stack for this task: Typescript, Javascript.
Libraries: Angular (base structure), big-integer (generate 'power' numbers), ngx-spinner (spinner).
A) Write a function that sorts 11 small numbers (<100) as fast as possible. Estimate how long it would take to execute that function 10 Billion (10^10) times on a normal machine?
    Key Points:
  • Quick sort (with random pivot, to get most efficient result) algorithm used. (fastest way for this task)
  • Average execution time (10B times) - 55 seconds.
  • setTimeout is used for show spinner while sorting function is executed.
B) Write a function that sorts 10000 powers (a^b) where a and b are random numbers between 100 and 10000? Estimate how long it would take on your machine?
    Key Points:
  • Native Array.sort is used (fastest way), also checked - base- 2^8 LSD radix sort, quicksort, but it was not effective.
  • Average execution time (only for sorting) - 0.009 seconds.
  • setTimeout is used for show spinner while generating/sorting function is executed.
  • Most of time on sorting is related on generating 'Power' array.
